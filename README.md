# homebrew-tap

Homebrew tap for [fredoliveira](https://github.com/fredoliveira)'s tools.

## Install

```sh
brew install fredoliveira/tap/seventeenlands-rust
```

## Formulae

- **seventeenlands-rust** — Rust port of the 17Lands MTGA log client.
  Source: https://github.com/fredoliveira/17lands-rust
